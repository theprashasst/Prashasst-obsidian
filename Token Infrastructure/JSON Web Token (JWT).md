# JSON Web Token (JWT)

## Definition

A JSON Web Token (JWT) is a compact signed token format used to transmit claims between parties.

## Key Ideas

- Consists of header, payload, and signature.
- Can be verified without database lookup.
- Widely used for API authentication.
